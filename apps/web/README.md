This is the public + admin web application for the Brouwerij Klein Zwitserland webshop. Built with Next.js 14 (App Router), Tailwind CSS, and TypeScript.

## Getting Started

First, run the development server from repo root:

```bash
npm run web:dev -w web
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `src/app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
